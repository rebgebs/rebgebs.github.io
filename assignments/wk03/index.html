Weather Site Name

The website name will be 'Wake Up To Weather'.

Weather Site Purpose

The purpose of this site is to prepare people for the day by informing them of the weather conditions in their area. Informing the user of the temperature, cloud cover, precipitation, wind speed, sunup, and sundown. The user will also find other useful information about the region like activities or group gathering that will allow the user to participate inthe community.
